# Memo Clock-记忆时钟 App Store Pages

这个文件夹用于发布 App Store 需要的两个公开 URL：

- `privacy.html`：隐私政策 URL
- `support.html`：支持 URL
- `privacy-en.html`：英文隐私政策 URL
- `support-en.html`：英文支持 URL

## 推荐发布方式

1. 新建一个 GitHub 仓库，例如 `memo-clock-pages`。
2. 把本文件夹里的所有文件上传到仓库根目录。
3. 进入仓库 `Settings > Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，Folder 选择 `/(root)`，点击 `Save`。
6. 等 GitHub Pages 发布完成。

如果你的 GitHub 用户名是 `yourname`，仓库名是 `memo-clock-pages`，最终 URL 通常是：

- 隐私政策：`https://yourname.github.io/memo-clock-pages/privacy.html`
- 支持页面：`https://yourname.github.io/memo-clock-pages/support.html`
- 英文隐私政策：`https://yourname.github.io/memo-clock-pages/privacy-en.html`
- 英文支持页面：`https://yourname.github.io/memo-clock-pages/support-en.html`

如果你创建的是用户站点仓库，例如 `yourname.github.io`，最终 URL 通常是：

- 隐私政策：`https://yourname.github.io/privacy.html`
- 支持页面：`https://yourname.github.io/support.html`
- 英文隐私政策：`https://yourname.github.io/privacy-en.html`
- 英文支持页面：`https://yourname.github.io/support-en.html`
