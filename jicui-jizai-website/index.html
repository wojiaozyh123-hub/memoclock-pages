Project Background

从“秸秆难处理”到“环保猫砂新供给”

农作物秸秆体量大、季节性强，传统处理方式容易带来资源浪费与环境压力；宠物消费升级又推动猫砂市场持续增长。秸萃净砂把两端需求连接起来，用微粉化技术提升秸秆吸水、结团和除臭表现，让农业副产物进入高频消费场景。

项目定位

面向新消费宠物用品市场，提供兼顾环保、体验与成本的秸秆基猫砂产品，并通过地方收储、加工转化和品牌销售形成可复制的助农闭环。

Core Technology

核心技术路径

01

秸秆微粉化

通过精细化粉碎和粒径控制，提高秸秆纤维吸水、包裹与成型能力，为猫砂性能打底。

02

生物胶粘剂

以绿色胶粘体系改善结团强度与崩解可控性，兼顾清理便利和可冲厕需求。

03

除臭与低尘

围绕异味吸附、粉尘控制和使用触感优化配方，让环保材料具备日常使用体验。
